# Portfolio Website

A responsive personal portfolio website that showcases my profile, skills, projects, resume, and contact information.

## 🌟Features

* Personal introduction and profile section
* About Me page
* Projects showcase
* Resume section
* Contact information page
* Responsive design using HTML and CSS
* Profile image and introduction video

## 📂 Project Structure

```text
Portfolio-website/
│
├── index.html          
├── about.html          
├── project.html        
├── resume.html         
├── contact.html       
├── style.css           
├── profile.webp        
├── my_intro.mp4        
├── background 1.jpg    
└── resume finall       
```

## 🛠️ Technologies Used

* HTML5
* CSS3

### Open the Project

1. Navigate to the project folder.
2. Open `index.html` in your web browser.

No additional installation or dependencies are required.

## 📸 Website Sections

### Home

Provides a brief introduction and navigation to other sections.

### About

Contains information about my background, skills, and interests.

### Projects

Displays the projects I have worked on and their details.

### Resume

Provides access to my resume and professional qualifications.

### Contact

Contains contact details and ways to connect with me.

## 🎯 Purpose

This portfolio website serves as a personal online presence to showcase my skills, projects, achievements, and professional profile.
